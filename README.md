// Proyecto Web Colaborativo

Este proyecto fue desarrollado por el grupo como parte del trabajo de creación de una página web estática usando GitHub de forma colaborativa.

---

// Estructura del proyecto

- index.php → Página principal
- about.php → Sobre nosotros
- contact.php → Página de contacto
- register.php → Formulario de registro
- css/estilos.css → Hojas de estilo
- js/funciones.js → Scripts
- img/logo.png y banner.jpg → Imágenes del sitio

---

 // Pasos del trabajo colaborativo (GitHub)

1. Crear repositorio principal.
2. Cada integrante desarrolla un módulo (index, about, contact o register).
3. Clonar el repositorio.
4. Aportar desde su carpeta local.
5. Enviar cambios usando:
   - git add .
   - git commit -m "mensaje"
   - git push
6. El líder revisa y acepta los cambios.
7. Enviar el enlace al docente y agregarlo como colaborador.
8. Presentar informe técnico del proyecto.

---

// Objetivo

Aprender trabajo colaborativo, manejo de repositorios y creación de una página web estática.
